### Вариант

**Лямбда-выражение в С++11**
```
int main() {
    auto square = [](int x) -> int { return x * x; };
    int a = 5;
    int b = square(a);
    return b;
}

```

### Установка clang и llvm

```
sudo apt install clang llvm
```

### Компиляция абстрактного дерева

```
clang++ -Xclang -ast-dump -fsyntax-only main.cpp
```
